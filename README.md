# 🛒 Sistema de Ventas MiniMarket

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)
![Cloud](https://img.shields.io/badge/Clever_Cloud-171E2D?style=for-the-badge&logo=clever-cloud&logoColor=white)

Un sistema de escritorio robusto desarrollado en Java para gestionar las operaciones diarias de un minimarket. Este proyecto centraliza el control de ventas, inventario y usuarios, conectándose en tiempo real a una base de datos alojada en la nube.

## ✨ Características Principales
* **Arquitectura en la Nube:** Base de datos MySQL desplegada en Clever Cloud para sincronización en tiempo real.
* **Gestión Administrativa:** Módulos para administrar productos, categorías, turnos de caja y métodos de pago.
* **Seguridad:** Control de accesos mediante inicio de sesión seguro.
* **Portabilidad:** Empaquetado completo con `jpackage`, lo que permite su ejecución sin necesidad de preinstalar Java en el equipo destino.

## 📦 Instrucciones de Instalación

Puedes descargar el programa listo para usar desde la sección de **[Releases]** a la derecha de este repositorio.

1. **Descarga el instalador:** Haz clic en el archivo `.exe` que se encuentra en los recursos (Assets).
2. **Inicia el asistente:** Haz doble clic sobre el archivo descargado para comenzar la instalación.
3. **Pasa las advertencias de Windows:** Al ser un software independiente desarrollado con fines académicos y sin una firma digital comercial, es totalmente normal que Windows active sus sistemas de prevención. Solo debes seguir estos sencillos pasos:

> 🛡️ **Nota de Seguridad Importante:**
> 
> * **Si aparece la pantalla de advertencia (UAC):** En la ventana que pregunta si deseas permitir que un *"editor desconocido"* realice cambios en el dispositivo, haz clic en **"Sí"**.
>
> _El programa es 100% seguro y libre de malware. Estas alertas aparecen únicamente porque el instalador fue compilado de manera local con jpackage._
